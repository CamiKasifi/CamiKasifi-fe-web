# Katkıda Bulunma Rehberi

CamiKasifi projesine katkıda bulunmak istediğiniz için teşekkür ederiz! 🙏

## 🚀 Başlarken

1. Projeyi fork edin
2. Yerel makinenize klonlayın
3. Bağımlılıkları yükleyin: `npm install`
4. Geliştirme sunucusunu başlatın: `npm run dev`

## 📝 Kod Standartları

- TypeScript kullanın
- ESLint kurallarına uyun
- Anlamlı commit mesajları yazın
- Her özellik için ayrı branch oluşturun

## 🔄 Pull Request Süreci

1. Değişikliklerinizi feature branch'inde yapın
2. Kodunuzu test edin
3. Anlaşılır bir PR açıklaması yazın
4. Review sürecini bekleyin

## 🐛 Bug Bildirimi

GitHub Issues üzerinden detaylı bug raporları oluşturun:
- Hatanın açıklaması
- Tekrar etme adımları
- Beklenen davranış
- Ekran görüntüleri (varsa)

## 💡 Özellik Önerileri

Yeni özellik önerileri için GitHub Issues kullanın ve aşağıdaki bilgileri ekleyin:
- Özelliğin detaylı açıklaması
- Kullanım senaryoları
- Mockup/tasarım (varsa)

## 📧 İletişim

Sorularınız için GitHub Discussions kullanabilirsiniz.

Teşekkürler! ❤️
